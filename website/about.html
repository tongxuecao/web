关于本站

所用技术

本网站使用纯前端技术搭建，未使用任何框架，适合初学者学习参考。

  • HTML：负责页面结构，使用语义化标签（header、main、form、table 等）
  • CSS：负责页面样式，使用 Flex 布局、圆角、阴影、过渡动画等
  • CSS 变量 / 单位：使用 vw、vh、rgba 等实现自适应和半透明效果
  • CSS 伪类：使用 :hover 实现鼠标悬停交互，无需 JavaScript

页面组成

  • 首页：四栏横向导航卡片，点击跳转各板块
  • 登录 / 注册页：表单页面，浅蓝色调，统一风格
  • 个人简历：悬停展示内容的四格卡片布局
  • 照片墙：S 形散落排列，悬停放大展示
  • 关于本站：即本页面

制作目标

  • 巩固 HTML 基础标签的使用，理解语义化结构
  • 掌握 CSS 常用布局方式，尤其是 Flex 弹性布局
  • 学会用 CSS 实现简单的交互效果，减少对 JavaScript 的依赖
  • 建立自己的个人网站，记录学习过程和生活点滴
