后端工程师职业规划

从入门到专家，四个阶段 · 清晰路径

01

入门期 · Junior

0 — 2 年 · 打好基础

学什么

  • 选一门语言：Java / Go / Python
  • 数据结构与算法基础
  • HTTP 协议 & RESTful API
  • SQL 数据库 + Redis 入门
  • Git 版本控制 & Linux 基础

目标

  • 能独立完成功能模块开发
  • 学会查日志、定位 Bug
  • 参与 Code Review，写单元测试
¥ 10k — 20k / 月
02

成长期 · Mid-Level

2 — 4 年 · 系统思维升级

学什么

  • 微服务架构 & Spring Boot
  • 消息队列：Kafka / RabbitMQ
  • Docker 容器化 & K8s 入门
  • CI/CD 流水线 & 性能调优

目标

  • 独立设计中等复杂系统
  • 输出技术方案文档
  • 开始带新人，参与技术选型
¥ 20k — 40k / 月
03

高级期 · Senior

4 — 7 年 · 架构 + 领导力

学什么

  • 分布式理论：CAP / BASE
  • 分布式事务：Saga / TCC
  • K8s 生产运维 & Service Mesh
  • 可观测性：日志 / 指标 / 链路

目标

  • 主导百万 QPS 系统架构设计
  • 带 3—5 人团队，制定技术规范
  • 推动服务 SLA ≥ 99.99%
¥ 40k — 70k / 月
04

专家期 · Staff / Principal

7 年以上 · 行业影响力

做什么

  • 自研核心基础组件与中间件
  • 主导或贡献开源项目
  • 制定 3 年技术战略规划
  • 发表技术论文 & 对外布道

目标

  • 某领域达到行业顶尖水平
  • 推动公司或行业标准演进
  • 建立个人技术品牌影响力
¥ 70k — 150k+ / 月
05

Senior 之后，选择你的方向

技术专家 IC

深耕技术
Senior → Staff → Principal
不做管理，靠技术影响公司

工程管理 EM

带团队做交付
TL → EM → Director → CTO
适合喜欢影响他人的人

创业 / 独立

做 CTO 或独立开发者
把技术变成产品和收入
适合有商业思维的人

海外 FAANG

进入全球顶级大厂
需备考英语 + 算法面试
TC 可达 $200k — $600k+
